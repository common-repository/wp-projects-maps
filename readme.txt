== Description ==
Great plugin for example Architects, Construction company's, Photographers etc. Who wants to show their projects on a map to highlight thee locations of their project.

1. Add the GEO location in your post
1. Using the Google maps API
1. Easy to use!

== Installation ==
1. Upload "/wp-projects-maps/" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Insert the Google Maps API key in the settings.
1. Place "[wpprojectsmaps width='' height='']" in your templates.
